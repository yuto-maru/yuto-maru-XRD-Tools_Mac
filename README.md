# XRD Tools

**XRD Tools** is a macOS application for X-ray diffraction (XRD) data
analysis.

## Features

-   Peak Picker (.chi) --- BL40B2
-   Peak Picker (.dat) --- BL19B2
-   Columnar Oblique Indexing
-   Columnar Rectangular Indexing
-   TIF Viewer
-   Excel → Word Table

## Installation

1.  Download the latest **XRD Tools.app** (or ZIP file) from the
    Releases page.
2.  Unzip the downloaded file if necessary.
3.  Drag **XRD Tools.app** to your Applications folder or any preferred
    location.

------------------------------------------------------------------------

## First-time Setup

Before using XRD Tools, install the required Python packages:

``` bash
pip install numpy pandas matplotlib scipy pillow pyperclip
```

Then launch **XRD Tools.app**.

If macOS warns that the app is from an unidentified developer:

-   Right-click the app and choose **Open**.
-   Click **Open** again in the dialog.

------------------------------------------------------------------------

## Updating

To update XRD Tools:

1.  Download the latest release.
2.  Replace your existing **XRD Tools.app** with the new version.

Your launcher settings (such as previously used folders) are preserved
automatically.

------------------------------------------------------------------------

## Version

The current version is shown at the bottom of the launcher window.
